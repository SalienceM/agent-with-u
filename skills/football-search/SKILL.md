# Football Search

搜索天下足球网的足球视频资源。

## 使用方法

使用 Bash 工具执行以下命令（Python 路径由安装环境自动填充）：

```bash
"<PYTHON>" .claude/skills/football-search/_call.py "<QUERY>" "<CATEGORY>"
```

## QUERY 构造规则（关键）

**不要直接使用用户原话。** 必须将用户意图拆解为足球领域的原始关键词，用**空格**分隔。

提取以下维度的关键词：

- **球队名**：使用中文常用简称，如 `皇马` `巴萨` `利物浦` `曼联` `曼城` `拜仁` `国米` `AC米兰` `切尔西` `阿森纳` `多特` `大巴黎` `尤文`
- **联赛/杯赛**：如 `英超` `西甲` `德甲` `意甲` `法甲` `欧冠` `欧联` `世界杯` `亚冠` `足总杯` `国王杯` `中超`
- **时间**：如 `2024` `2025` `24-25赛季`
- **球员名**：如 `梅西` `C罗` `姆巴佩` `哈兰德` `维尼修斯` `贝林厄姆`
- **轮次/阶段**：如 `决赛` `半决赛` `第20轮` `小组赛`

### 示例

| 用户说的 | QUERY | CATEGORY |
|---------|-------|----------|
| 我想看昨天皇马对巴萨的比赛录像 | `皇马 巴萨` | `replay` |
| 有没有2024欧冠决赛集锦 | `欧冠 决赛 2024` | `highlight` |
| 找一下梅西在国际迈阿密的进球 | `梅西 国际迈阿密` | （不填） |
| 最近英超曼城的比赛 | `英超 曼城` | （不填） |
| 今晚有什么球赛直播 | `直播` | `live` |
| 利物浦阿森纳英超 | `利物浦 阿森纳 英超` | （不填） |

## 参数说明

| 参数 | 必填 | 说明 |
|------|------|------|
| QUERY | 是 | 空格分隔的足球关键词组合，每词 2-4 字，总共不超过 5 个词 |
| CATEGORY | 否 | `replay`=录像　`highlight`=集锦　`live`=直播，不填则搜索全部类型 |

## 规则

- **只执行一次**，将输出直接原样呈现给用户
- 不要重试、不要自行判断结果质量、不要额外处理输出
- 关键词保持简短，每个词 2-4 个字为佳，总共不超过 5 个词
