# Football Search

搜索天下足球网的足球视频资源。

## 使用方法

使用 Bash 工具执行以下命令（Python 路径由安装环境自动填充）：

```bash
"<PYTHON>" .claude/skills/football-search/_call.py "<QUERY>" "<CATEGORY>"
```

## QUERY 构造规则（关键）

**用最少的词命中最多的结果。** 搜索引擎压力大、精确词反而漏结果，因此关键词**越少越好**。

### 核心原则

1. **默认只用 1 个词**：联赛名或球队名，用户自行筛选结果
2. **最多 2 个词**：仅当用户明确指定了两方（如两支球队对阵）才加第二个词
3. **不加时间/赛季**：除非用户要找某年的特定决赛
4. **不加轮次/阶段**：用户可以自己看标题筛选

### 示例

| 用户说的 | QUERY | CATEGORY |
|---------|-------|----------|
| 找欧冠视频 | `欧冠` | （不填） |
| 想看皇马对巴萨的录像 | `皇马 巴萨` | `replay` |
| 找梅西的进球集锦 | `梅西` | `highlight` |
| 最近英超曼城的比赛 | `曼城` | （不填） |
| 今晚有什么球赛直播 | `直播` | `live` |
| 2024欧冠决赛集锦 | `欧冠 决赛` | `highlight` |
| 看看哈兰德的进球 | `哈兰德` | （不填） |

## 参数说明

| 参数 | 必填 | 说明 |
|------|------|------|
| QUERY | 是 | **1 个词为主，最多 2 个词**；每词 2-4 字，不加时间不加轮次 |
| CATEGORY | 否 | `replay`=录像　`highlight`=集锦　`live`=直播，不填则搜索全部类型 |

## 输出规则（严格遵守）

⚠️ **禁止对输出做任何加工**：不得重新排版、不得重写标题、不得合并行、不得翻译或修正任何字段。

⚠️ **只执行一次**：无论结果是否有乱码、是否理想，都不重试，不换关键词再试。

执行后将结果按如下格式直接呈现给用户：

```
## 搜索结果：<query>（共 N 条）

1. <title>
   🔗 <url>
   📅 <date>

2. ...
```

- **必须展示 url**，用户需要通过链接访问帖子获取下载地址
- 输出为空（count=0）时，如实告知用户无结果
- 输出报错（ok=false）时，直接告知用户错误信息，不尝试其他关键词
